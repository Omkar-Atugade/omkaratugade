LIFE IN GAME

About project :

This project is about health potion in games. If game has three difficulty levels, according to the difficulty level you have selected the amount of health the player gets from the potion will change. Health potion will be less in high difficulty level and more in less difficulty level.


In this project i have used variables and numbers to create a functionality required for health potion to work and i have also used pythons random module to build random number generator.
