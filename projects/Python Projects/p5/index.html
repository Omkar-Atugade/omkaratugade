CINEMA SIMULATOR

About project :

In this project, the program let the user choose from range of films. When user will pick a film, program will ask them for thier age. If they are old enough to watch the film and their enough seats are available then the program will let the user to book to see the film.


If not, the program will tell them that they are too young to see the film or their are not enough seats available. I have used variable, logic and data structure concepts in python to build this cinema simulator.
