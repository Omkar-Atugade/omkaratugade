SNAKE GAME

About Game :

Snake game is the game in which we have snake and we haev eat food by moving the sanke in given boundary using up, down, right, left keys. The game is about eating more and more number of food items without hitting the boundary of window provided as well as without hitting the snake body itself.


It is game build using javascript. I have created canvas using html and script is added to the same which conatins all the js code.
